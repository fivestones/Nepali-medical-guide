# Part 1: Navigating Sensitive Language and Clinical Registers

Nepali culture leans conservative regarding bodily functions, sexual health, and anatomy. However, medical professionals must gather specific information. Use this framework to navigate sensitive topics:

- **The "Funnel" Approach:** Always start with a respectful, broad euphemism to build trust. Only narrow down to highly specific/clinical terms when medically necessary.

  - *Example:* Start by asking if they have pain in their *Guptanga* (गुप्ताङ्ग - private parts). If they say yes, explain *why* you need to be specific: "Malaai rog patta lagauna tapai-ko samasya thakkai kahan chha bhannu parchha" (मलाई रोग पत्ता लगाउन तपाईँको समस्या ठ्याक्कै कहाँ छ भन्नु पर्छ।) (To diagnose the disease, you need to tell me exactly where your problem is). Then use the clinical term.

- **Understand the Registers:**

  - **Formal/Medical:** Used in charting, formal diagnoses, and showing professional respect.

  - **Colloquial Adult:** The everyday language patients will use to describe their symptoms to you.

  - **Pediatric/Child Talk:** Crucial for pediatricians or when speaking to mothers about their infants.

  - **Taboo/Vulgar Slang:** You should generally avoid using these, but you *must* understand them if a patient uses them.

- **Tone & Respect:** Always use *Tapai* (तपाईँ - formal "you") and the *Hajur* (हजुर) register when discussing sensitive issues. It creates a clinical distance that helps Nepali patients feel more comfortable and respected.

- **A Note on Medical Verbs (Tense Modifiers):** Nepali verbs change heavily based on tense. As a practitioner, you will hear a root verb modified in three common ways. For example, using the root *Kaam-* (काम-; to shake/tremble):

  - **Present (It happens generally/now):** *-cha* suffix -> *Kaamcha* (काम्छ - It shakes).

  - **Past (It happened):** *-yo* suffix -> *Kaamyo* (काम्यो - It shook).

  - **Perfect/State (It has happened/is in a state of):** *-eko* suffix -> *Kaameko* (कामेको - It is shaking / has shaken).

  - *Example with Pain (Dukh-* / दुख-*):* *Dukhcha* (दुख्छ - It hurts), *Dukhyo* (दुख्यो - It hurt), *Dukheko chha* (दुखेको छ - It is currently hurting).

## Quick Register Table for Bodily Functions

This table is a fast reference for common bodily functions where register matters. Use the formal and colloquial columns actively; the taboo column is here for recognition, not for clinician-facing speech.
